SOLO LA PRIMERA VEZ EN UNA PC NUEVA (ya hecho en esta PC)
----------------------------------------------------------
1. Instalar Git:        https://git-scm.com
2. Instalar GitHub CLI: winget install --id GitHub.cli
3. Autenticarse:        gh auth login
   - Elegir: GitHub.com -> HTTPS -> Login with a web browser
   - Se abre el navegador, aceptar con la cuenta eugenioorosco8@gmail.com


CADA VEZ QUE MODIFIQUES ARCHIVOS Y QUIERAS SUBIR
----------------------------------------------------------
1. Abrir PowerShell o CMD (Win + R, escribir "powershell", Enter)

2. Ir a la carpeta de ejemplos:
   cd "C:\Users\54264\OneDrive\ProfTitular\DocApuntes\MSP430\Claude\ejemplos"

3. Ver que cambio (opcional, para saber que vas a subir):
   git status

4. Preparar todos los cambios:
   git add .

5. Guardar un snapshot con descripcion:
   git commit -m "descripcion breve de lo que cambiaste"

   Ejemplos de mensajes:
     git commit -m "corregi PMM en DEM_G3_E3"
     git commit -m "agregue ejemplo GL_U5_04"
     git commit -m "fix volatile en archivos ADC"

6. Subir a GitHub:
   git push
